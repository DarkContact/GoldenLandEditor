# GoldenLandEditor
Редактор уровней для игр Златогорье 2 и Холодные небеса (GoldenLand 2, ColdHeaven)
![preview](https://github.com/user-attachments/assets/d5f2b0d5-69cc-4ddb-a876-13e9e90073c7)

## Возможности
- Просмотр уровней
- Просмотр CSX файлов (графика)
- Просмотр SDB файлов (базы строк)
- Просмотр MDF файлов (магические эффекты)
- Просмотр и исполнение CS файлов (скомпилированные диалоги)

## Готовые сборки
Находятся в разделе [Releases](https://github.com/DarkContact/GoldenLandEditor/releases)

## Сборка проекта (Windows)
Проект собирается с помощью компиляторов **MSVC** или **MinGW** с поддержкой **C++20**  
Для сборки проекта требуется **SDL3**, его можно скачать через скрипт `get_SDL3.py` в директории `external`
