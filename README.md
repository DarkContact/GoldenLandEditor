### Настройка проекта (Windows)
Проект собирается с помощью компиляторов **MSVC** или **MinGW** с поддержкой **C++20**  

Для сборки проекта требуется **SDL3**  
Есть несколько способов указать путь к библиотеке:
1. Скачать **SDL3** через скрипт `get_SDL3.py` в директории `external`
2. Установить переменную `SDL3_DIR`, указывающую на директорию, содержащую `SDL3Config.cmake`
3. Установить переменную `CMAKE_PREFIX_PATH`, указывающую на корневую директорию установки **SDL3**
4. Использовать **vcpkg**
