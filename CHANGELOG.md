## [0.60.0] - 2025-12-19

### Что нового
#### Общее
- Пристыковка окон к главной ноде по умолчанию, не создаём imgui.ini
- Отображение и взаимодействие с диалогом в CsExecutor
- Отображение прогресса выполнения инструкций диалога в CsViewer
- Отображение триггеров на уровнях
- Добавлена иконка и описание для файла редактора
- Встроен шрифт Carlito (Используется по умолчанию)

#### Техническое
- Появление функции formatToBuffer - позволяет избегать динамических аллокаций памяти
- Добавлен удобный механизм встраивания ресурсов в exe файл
- Добавлены методы в CsViewer для встраивание озвучки в диалоги

#### SdbViewer
- Добавлен выбор фильтра по тексту или ID 
- Добавлен показ символов форматирования в строке
- Добавлен тултип для удобного чтения длинных текстов (При зажатии ЛКМ)
- Добавлена возможность копировать текст (При нажатии ПКМ)


### Исправления
- Корректная отрисовка тайлов в режиме отображения масок ("Tiles" -> "Tiles mode" -> "Mask")
- Оптимизация неотображаемых уровней (не считают свою логику)
- Оптимизация отрисовки больших таблиц в SdbViewer
- Убраны лишние аллокации для строк (SdbViewer, CS_Node)
